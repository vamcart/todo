## Установка и настройка

Откройте `config/app.php` и измените доступ к базе данных mysql.

1. Создайте указанную в конфиге базу данных, загрузите в базу todos.sql
2. Перейдите в корневую папку (там где этот файл находится).
3. Выполните команду `composer install`
4. Запустите веб-сервер `php -S localhost:8000 -t public`
5. Откройте в браузере [`http://localhost:8000`](http://localhost:8000)
